Ici on les diffirents fichier qui nous permettent de lancer le jeu:

* dice.py: contient une fonction qui renvoie juste un nombre entre 1 et 6 (représente un lancer de De).
* hog.py contient les différentes fonctions de Jeu et les strategies.
* hog_gui.py contient toutes les fonctionalite grpahique de jeu sequentielle.
* outils.py contient toutes les fonctions qui nous permettent de calculer les espirances de gain et la resolution des PL.
* main_seq.py c'est la fonction main pour lancer le jeu sequentielle on utilise le module tkinter.
* main_simu.py c'est la fonction main pour lancer le jeu simultane on utilise le module PyQt5.
