Ici on les diffirents fichier qui nous permettent de lancer le jeu:

1: dice.py: contient une fonction qui renvoie juste un nombre entre 1 et 6 (représente un lancer de De).
2: hog.py contient les différentes fonctions de Jeu et les strategies.
3: hog_gui.py contient toutes les fonctionalite grpahique de jeu sequentielle.
4: outils.py contient toutes les fonctions qui nous permettent de calculer les espirances de gain et la resolution des PL.
5: main_seq.py c'est la fonction main pour lancer le jeu sequentielle on utilise le module tkinter.
6: main_simu.py c'est la fonction main pour lancer le jeu simultane on utilise le module PyQt5.
